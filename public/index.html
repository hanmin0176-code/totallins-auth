허브로 돌아가기 API 상태 확인

Totallins Auth

공통 로그인, 세션, 초기 활성화, 계정 운영을 담당하는 중앙 인증 서비스입니다.

직원 계정 최초 활성화는 이제 auth에서 처리하고, 운영 계정 관리는 별도 운영센터에서 관리합니다.

운영센터 열기 초기 활성화 페이지 허브 로그인

인증 API

허브와 명함 서비스는 아래 API를 통해 동일한 세션 쿠키를 사용합니다.

  • POST /api/auth/login
  • GET /api/auth/me
  • POST /api/auth/logout
  • POST /api/auth/activate

운영 API

계정 등록과 권한 관리는 card가 아니라 auth를 기준 원장으로 사용합니다.

  • GET /api/admin/dashboard
  • POST /api/admin/employee-import
  • POST /api/admin/employee-reset-password
  • POST /api/admin/employee-active
  • POST /api/admin/employee-update
  • POST /api/admin/employee-delete
